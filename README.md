📌 Overview  
This project demonstrates manual testing of a University Management System (UMS), a web-based application used to manage academic activities such as student enrollment, attendance, and result processing.





 🧩 Modules Tested  
- Authentication (Login/Registration)  
- Course Enrollment  
- Attendance Management  
- Result System  
- Payment System  



 🧪 Testing Types  
- Functional Testing  
- UI Testing  
- Regression Testing  
- Negative Testing
